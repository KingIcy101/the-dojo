# Shadcnblocks — Pre-built Page Sections

## What It Is
shadcnblocks.com — library of complete page sections built on shadcn/ui + Tailwind.
Copy the code directly into your project. You own it, customize it freely.

## Key Sections Available
- **Hero** — multiple layouts (centered, split, with video, with mockup)
- **Features** — grid, alternating, icon-based
- **Pricing** — monthly/annual toggle, tier cards
- **Testimonials** — carousel, grid, quotes
- **FAQ** — accordion, two-column
- **CTA** — centered, banner, with email capture
- **Footer** — multi-column, minimal
- **Navigation** — mega menu, simple, with auth

## Workflow
1. Go to shadcnblocks.com
2. Find the section you need
3. Copy the component code
4. Drop into your project and customize

## Rules
- Always customize colors/fonts to match the design system — don't use default shadcn gray palette
- Remove any hardcoded text immediately and replace with real copy
- Check mobile layout after pasting — most blocks are responsive but verify
- Use as a starting point, not final output — always add brand identity on top

## Best Blocks for In The Past AI / Agency Sites
- Hero with email capture + social proof numbers
- Features with alternating image+text layout
- Testimonials carousel with avatar + quote
- Pricing with monthly/annual toggle
- FAQ accordion
